# Резервы

## Для чего нужен раздел

Резервы показывают, сколько валюты обменник готов выдать клиентам. Резерв влияет на доступность направлений и максимальную сумму заявки.

Для полного разбора обычных резервов, связанных резервов, истории изменений и резервов из файлов используйте страницу [Резервы: обычные, связанные и из файлов](reserves-reference.md).

## Когда использовать

Открывайте резервы, когда нужно изменить доступную сумму, привязать резерв к файлу, сделать резерв фиксированным, проверить историю изменений или понять, почему направление не дает создать заявку на большую сумму.

## Основные возможности

| Возможность | Что означает простыми словами | Когда использовать |
| --- | --- | --- |
| Список резервов | Резерв по валютам | Чтобы изменить доступную сумму |
| Фиксированный резерв | Сумма задается вручную и не зависит от автоматических источников | Когда резерв нужно держать строго заданным |
| Резерв из файла | Значение подтягивается из файла | Когда остатки приходят от внешней системы |
| Связанный резерв | Одна валюта использует резерв другой | Когда несколько валют должны делить общий лимит |
| История резервов | Кто и когда изменил сумму | Когда нужно проверить спорную ситуацию |
| Группы файлов резервов | Организация файловых источников | Когда есть несколько поставщиков резервов |

## Поля и настройки

| Поле / настройка | Что означает | Как правильно заполнять | Важно знать |
| --- | --- | --- | --- |
| Валюта | К какой валюте относится резерв | Выберите правильную валюту и сеть | Ошибка влияет на доступность направления |
| Сумма | Доступный резерв | Укажите фактическую доступную сумму | Слишком высокий резерв может принять невозможную заявку |
| Фиксированный резерв | Не менять резерв автоматически | Включайте для ручного контроля | Автообновление не будет корректировать сумму |
| Резерв из файла | Источник автоматической суммы | Выберите файл только после проверки | Неверный файл даст неверный резерв |
| Использовать резерв | Привязка к другому резерву | Используйте для общих остатков | Цепочки резервов нужно проверять внимательно |
| Комментарий | Объяснение изменения | Заполняйте причину ручной правки | Помогает в истории и поддержке |

## Как настроить

1. Откройте "Основное" и перейдите в "Резервы".
2. Найдите нужную валюту.
3. Откройте редактирование резерва.
4. Укажите сумму или выберите файл резерва.
5. Если резерв должен быть неизменным, включите фиксированный резерв.
6. Если несколько валют используют один общий остаток, выберите связанный резерв.
7. Добавьте комментарий к изменению.
8. Сохраните и проверьте направление, где используется эта валюта.

## Типовые ошибки

| Ошибка / ситуация | Причина | Что сделать |
| --- | --- | --- |
| Направление не принимает сумму | Резерв меньше суммы заявки | Увеличить резерв или изменить лимит направления |
| Резерв сам меняется | Используется файл или связанный резерв | Проверить источник резерва |
| На сайте показан неверный остаток | Включен кеш или файл резерва устарел | Обновить файл и очистить кеш при необходимости |
| История показывает неожиданное изменение | Резерв изменил другой менеджер или автоматический источник | Проверить историю резервов и источник |
| Сумма не уменьшается после заявки | Настроен фиксированный резерв или неверная логика списания | Проверить тип резерва и связанные настройки |

## Важные предупреждения

Не завышайте резерв вручную, если фактических средств нет. Это может привести к заявкам, которые обменник не сможет выполнить.

Перед включением связанного резерва проверьте, какие валюты и направления будут использовать общий остаток.
