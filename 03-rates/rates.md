# Курсы и источники курсов

## Для чего нужен раздел

Раздел курсов отвечает за то, откуда обменник берет курс, как применяет корректировки и как пересчитывает суммы. Курс может быть ручным, взятым из источника, рассчитанным формулой, полученным из файла, курса конкурента или BestChange.

## Когда использовать

Открывайте раздел курсов, когда сумма в заявке считается неправильно, курс перестал обновляться, нужно подключить новый источник, изменить формулу, настроить BestChange или проверить историю изменений курса.

## Основные возможности

| Возможность | Что означает простыми словами | Когда использовать |
| --- | --- | --- |
| Источники курсов | Внешние или внутренние источники, откуда берется курс | Когда нужно автоматическое обновление |
| Ручной курс | Значение, введенное администратором | Когда курс нужно зафиксировать вручную |
| Корректировка | Надбавка или скидка к исходному курсу | Когда нужно добавить маржу или отступ от рынка |
| Формулы | Расчет курса из нескольких значений | Когда обычного источника недостаточно |
| Курсы из файла | Импорт значений из файла | Когда курс поставляет внешний файл |
| Курсы конкурентов | Ориентация на данные другого обменника | Когда нужно следить за рынком |
| BestChange API | Получение рыночных данных BestChange | Когда курс должен считаться по позиции, медиане или среднему |
| Аналитика курса | График и сводка по источнику | Когда нужно понять, скачет ли курс |

## Поля и настройки

| Поле / настройка | Что означает | Как правильно заполнять | Важно знать |
| --- | --- | --- | --- |
| Источник курса | Откуда брать базовый курс | Выберите один подходящий источник | Если источник не обновляется, направление получит старый курс |
| Добавить к курсу | Процентная или числовая корректировка | Указывайте понятную надбавку, например 0.5 | Ошибка знака меняет выгоду обмена |
| Ручной курс | Фиксированное значение | Используйте временно или для редких пар | Может не обновляться автоматически |
| Формула | Правило расчета курса | Проверяйте формулу на тесте | Ошибка формулы влияет на все заявки по направлению |
| Позиция BestChange | Номер строки в списке предложений | Укажите позицию или диапазон | Если строк меньше, система берет доступную позицию |
| Top N | Сколько лучших курсов учитывать | Обычно используют 3-10 | Слишком маленькое число делает курс резким |
| Медиана | Берет устойчивое значение из нескольких курсов | Используйте для сглаживания скачков | Не всегда дает самый выгодный курс |
| Взвешенное среднее | Учитывает курсы и резервы обменников | Используйте для более стабильного рынка | Требует корректных резервов в источнике |
| Черный список обменников | Кого не учитывать в BestChange | Добавляйте ненадежные обменники | Слишком большой список исказит рынок |
| Белый список обменников | Использовать только выбранных | Заполняйте только при четкой стратегии | Если список пустой, используется общий список |

## Как настроить

1. Откройте направление обмена.
2. Перейдите во вкладку курса.
3. Выберите один основной источник: ручной курс, источник, формула, файл, конкурент или BestChange.
4. Укажите корректировку, если нужно отступить от рынка.
5. Если используется BestChange, выберите стратегию: позиция, медиана или взвешенное среднее.
6. Проверьте, что выбранная пара соответствует формату "BTC - USDT" и правильной сети.
7. Сохраните настройки.
8. Создайте тестовую заявку и сравните сумму с ожидаемой.
9. Проверьте лог обновления курсов, если значение не изменилось.

## Типовые ошибки

| Ошибка / ситуация | Причина | Что сделать |
| --- | --- | --- |
| Курс не обновляется | CRON выключен или источник недоступен | Проверить CRON, источник и журнал парсера |
| Курс отличается от BestChange | Выбрана позиция, фильтр или корректировка | Проверить стратегию, черный список и шаг |
| Формула возвращает ошибку | Неверный тег или математическое выражение | Проверить формулу через тестовый запрос |
| Файл курсов не применяется | Файл не обновился или неверная группа | Проверить источник файла и группу |
| Курс конкурента пустой | Конкурентский источник не спарсился | Проверить ссылку конкурента и журнал |

## Важные предупреждения

Перед изменением формулы, стратегии BestChange, позиции, Top N и корректировки проверьте расчет на тестовом направлении. Неверный курс может привести к убыточным заявкам.

Не используйте одновременно несколько источников без понимания, какой источник выбран в конкретном направлении.
