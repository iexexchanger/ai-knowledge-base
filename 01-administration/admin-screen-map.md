# Карта экранов админ-панели

Этот файл нужен GitBook AI как навигационная карта. Если пользователь спрашивает “где это настроить”, AI должна сначала определить экран, затем открыть подробный reference-файл по полям.

Если вопрос шире одного экрана, например "что делать первым", "как устроена вся админка", "какие разделы связаны между собой" или "куда идти по симптомам", используйте также страницу "Полная карта админ-панели". Она связывает реальные разделы меню с порядком настройки, типовыми зависимостями и диагностикой.

## Главная логика админки

| Блок меню | Что управляет | Где смотреть подробности |
|---|---|---|
| Рабочий стол | Общие показатели, виджеты, быстрый поиск, уведомления | Панель управления, Диагностические сценарии |
| Основное | Валюты, направления, реквизиты, резервы, платёжные системы | Валюты, Направления обмена, Справочник валют, Справочник направлений |
| Заявки | Список заявок, карточка заявки, статусы, этапы, архив, лимиты операторов | Заявки, Операции, статусы и действия оператора |
| Верификация | Проверка карт и личности KYC | KYC и верификация, Справочник KYC и AML |
| Шлюзы | Мерчанты, автовыплаты, журналы платежных шлюзов | Платежные системы, Платежи, автовыплаты и уведомления |
| Парсер курсов | Источники, BestChange, конкуренты, формулы, файлы | Курсы, Источники курсов, формулы и резервы |
| Пользователи | Клиенты, роли, блокировки, доп. поля, скидки, уровни, бонусный центр | Пользователи, Справочник пользователей, ролей и доступа |

| Инструменты | AML/KYC-сервисы, прокси, города, черный список, API, отзывы, рабочий статус, BIN, чекбоксы | Справочник KYC и AML, Контент, маркетинг и инструменты |

## Карточка валюты

| Вкладка | Что настраивает | Когда открывать |
|---|---|---|
| Основные настройки | Статус, платёжная система, код валюты, XML, сеть, формат суммы | Валюта не видна, неверный код, нужна сеть |
| Поля | Поля реквизитов, маски, валидаторы, QR, доп. поля | Клиент вводит неверные реквизиты или не видит нужное поле |
| Информация | Описания, инструкции, предупреждения, кнопки, цвет | Нужно изменить текст на форме обмена |
| Верификация карты | Требование проверки карты по сумме | Для направлений с банковскими картами |
| Верификация личности | Требование KYC по валюте | Для валют с обязательной проверкой личности |
| Мерчанты | Какие мерчанты могут выдавать реквизиты для валюты | Реквизиты не выдаются или нужен другой провайдер |
| Выплаты | Какие автовыплаты доступны по валюте | Автовыплата не уходит или нужно ограничить провайдера |
| AML | Проверка кошелька или транзакции | Нужно включить риск-проверку |
| Лимиты резерва | Публичное отображение и ограничения резерва | Резерв отображается неверно |
| Реквизиты | Связи с ручными реквизитами | Для ручной выдачи счетов |
| Пересчёт | Автопересчёт заявок по валюте | Клиент оплатил позже или курс изменился |
| Партнёрка | Партнёрские настройки валюты | Если бонусы зависят от валюты |
| Опции | QR, файлы, email-верификация, этапы заявки | Когда нужно включить доп. сценарии |

## Карточка направления

| Вкладка | Что настраивает | Когда открывать |
|---|---|---|
| Основные | Статус, валюты, техническое название, min/max | Направление не видно или не создаёт заявку |
| Информация | Описание, инструкции, SEO, текст подтверждения | Нужно изменить текст для клиента |
| Поля | Доп. поля направления и пользователя | Нужно спросить город, банк, документ или комментарий |
| Верификация карты | Правила проверки карты именно для пары | Верификация нужна только в конкретном направлении |
| Верификация личности | KYC именно для пары | KYC нужен только для определённой пары |
| Города | Городские ограничения и комиссии | Для cash-направлений |
| Курсы | Источник курса, ручной курс, добавка, формула | Неверный курс или нужно подключить источник |
| BestChange | Настройка пары BestChange | Курс берётся от позиции или top N |
| Кратность | Сумма должна быть кратной значению | Для наличных, купюр, банковских требований |
| Ограничения | Доп. ограничения по направлению | Для лимитов и условий |
| Прибыль | Процент прибыли обменника | Для маржи |
| Комиссии | Дополнительные комиссии и комиссии платёжной системы | Для fees на стороне отдаю или получаю |
| Суммы обмена | Надбавки или комиссии по диапазонам суммы | Когда курс зависит от объёма |
| Fee selector | Выбираемые клиентом комиссии | Для speed fee, network fee, service fee |
| Резервы | Связь направления с резервами | Если резерв пары отличается от валютного |
| Реквизиты | Ручные реквизиты пары | Если реквизиты индивидуальны для направления |
| Пересчёт | Пересчёт заявки по направлению | При плавающем курсе и поздней оплате |
| Партнёр | Партнёрские настройки направления | Если вознаграждение зависит от пары |
| Экспорт | Вывод направления в экспорт курсов | Для XML, мониторингов и партнёров |
| Проверки ограничений | Диагностика, почему направление недоступно | Если сайт не показывает пару |
