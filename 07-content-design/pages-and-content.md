# Страницы и контент

## Для чего нужен раздел

Раздел нужен для управления публичными страницами сайта: меню, новостями, FAQ, страницами, преимуществами, контактами и статистическими блоками. Через него администратор меняет тексты, которые видят клиенты на сайте, в личном кабинете и на отдельных информационных страницах.

## Когда использовать

Открывайте этот раздел, когда нужно изменить текст на сайте, добавить новость, обновить FAQ, создать страницу с правилами, добавить контакт, изменить блок преимуществ или скрыть устаревшую информацию.

## Основные возможности

| Возможность | Что означает простыми словами | Когда использовать |
|---|---|---|
| Меню | Ссылки и пункты навигации сайта | Когда нужно добавить пункт “Правила”, “AML”, “Контакты” или изменить порядок меню |
| Страницы | Отдельные текстовые страницы сайта | Для правил обмена, политики, инструкций, пользовательских соглашений |
| Группы страниц | Категории для страниц | Когда страниц много и их нужно разделить по темам |
| Новости | Публикации на сайте | Для объявлений, обновлений, изменений графика работы |
| Категории новостей | Темы новостей | Если новости нужно разделить по типам |
| FAQ | Частые вопросы и ответы | Чтобы клиенты находили ответы без обращения в поддержку |
| Категории FAQ | Группы вопросов | Для разделения вопросов по темам: заявки, верификация, курсы, резервы |
| Преимущества | Блоки доверия на главной странице | Для описания сильных сторон обменника |
| Контакты | Контактные данные и каналы связи | Когда меняется Telegram, email, график поддержки или адрес |
| Группы контактов | Разделение контактов по смыслу | Например, “Поддержка”, “Партнёрам”, “Юридическая информация” |
| Статистика | Публичные показатели сайта | Для отображения количества обменов, клиентов, резервов, отзывов |

## Поля и настройки

| Поле / настройка | Что означает | Как правильно заполнять | Важно знать |
|---|---|---|---|
| Название | Заголовок страницы, новости, FAQ или пункта меню | Пишите коротко и понятно | Название часто видно клиенту |
| Статус | Показывать или скрывать запись | Включайте только проверенный контент | Отключённая запись обычно не видна на сайте |
| Текст / описание | Основной текст страницы или ответа | Пишите простым языком, без лишней технической терминологии | Не вставляйте секретные данные, реквизиты админов или внутренние ссылки |
| Категория | Группа, к которой относится запись | Выберите подходящую категорию | Неверная категория усложняет поиск клиентам |
| Сортировка | Порядок вывода на сайте | Чем важнее пункт, тем выше его нужно поставить | После изменения сортировки проверьте публичную страницу |
| Ссылка | URL или путь пункта меню | Используйте понятные человекочитаемые адреса | Не указывайте приватные URL админки |
| Изображение | Картинка новости, преимущества или баннера | Загружайте оптимизированные изображения | Слишком тяжёлые файлы замедляют сайт |
| SEO title | Заголовок для поисковых систем | Укажите короткий заголовок страницы | Влияет на отображение страницы в поиске |
| SEO description | Описание страницы для поисковых систем | 1–2 предложения о содержании страницы | Не должно быть длинным и рекламно-перегруженным |
| Кнопка / ссылка | Куда ведёт пользователь после клика | Указывайте только рабочие публичные ссылки | После сохранения проверьте переход |
| Язык | Текст для конкретной локали | Заполняйте все включённые языки | Если язык включён, пустой перевод может выглядеть плохо на сайте |

## Как настроить

1. Откройте нужный подраздел: Меню, Страницы, Новости, FAQ, Преимущества, Контакты или Статистика.
2. Найдите существующую запись или нажмите создание новой записи.
3. Заполните название, статус, категорию, текст и сортировку.
4. Для многоязычного сайта заполните тексты на всех включённых языках.
5. Если запись ведёт на страницу или внешний ресурс, проверьте ссылку.
6. Сохраните запись.
7. Откройте публичный сайт и проверьте, как запись выглядит для клиента.
8. Если включено кеширование контента, очистите кеш в настройках оптимизации.

## Типовые ошибки

| Ошибка / ситуация | Причина | Что сделать |
|---|---|---|
| Изменения не видны на сайте | Включён кеш или запись выключена | Проверить статус записи и очистить кеш |
| Пункт меню ведёт не туда | Неверно указана ссылка | Исправить URL и проверить переход |
| FAQ не отображается | Выключена категория или сам вопрос | Включить категорию и запись FAQ |
| На другом языке пустой текст | Не заполнена локаль | Заполнить перевод или временно отключить язык |
| Картинка не отображается | Файл удалён, путь неверный или формат не поддержан | Загрузить изображение заново |
| Текст выглядит слишком длинным | Включён блок с ограничением строк или текст без структуры | Разбить текст на абзацы и проверить настройки “Читать далее” |

## Важные предупреждения

Не добавляйте в публичный контент API-ключи, токены, приватные ссылки, внутренние инструкции операторов, персональные данные клиентов или реквизиты, которые не должны быть доступны всем. Изменения в правилах обмена, AML, KYC, возвратах и комиссиях лучше согласовывать с владельцем обменника, потому что клиенты могут ссылаться на эти тексты в спорных ситуациях.

Если меняете страницу с правилами, проверьте связанные чекбоксы соглашений в форме создания заявки. Клиент должен соглашаться именно с актуальной версией правил.
